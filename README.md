# 🎬 Netflix Analytics Dashboard

A Flask-based Movie Recommendation and Analytics Dashboard that provides personalized movie recommendations using Machine Learning and Cosine Similarity.

## 🚀 Live Demo

https://netflix-analytics-dashboard-hlgr.onrender.com

---

# 📌 Features

* 🎥 Movie Recommendation System
* 📊 Analytics Dashboard
* 🔍 Movie Search with Autocomplete
* 🧠 Recommendation using Cosine Similarity
* 🗂 Recent Search History
* 📈 Genre Analytics Chart
* 🖼 Movie Posters Display
* 💾 SQLite Database Integration

---

# 🛠 Technologies Used

* Python
* Flask
* Pandas
* Scikit-learn
* SQLite
* HTML
* CSS
* Chart.js

---

# 🧠 Machine Learning Concept

The recommendation system uses:

* TF-IDF Vectorizer
* Cosine Similarity

to recommend movies based on genre similarity.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Nireeksha125/Netflix-Analytics-Dashboard.git
```

## Open Project Folder

```bash
cd Netflix-Analytics-Dashboard
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```
